Create menu (Nav.jsx) which contains Group Chat | Manage Users | Manage Documents | Logout as per project wireframe 

When user click on Group Chat call ChatList component 
When user click on Manage Users call UserList component 
When user click on Manage Documents call DocumentList component

By default Welcome component 

Step 1
npm i react-router-dom

Step 2
Create Nav Component, Nav.jsx which contains below code

import { Link, Outlet } from 'react-router-dom';

// Home | AddProduct | Products | Title | Register | Bootstrap
function Nav() {
return (<>
<nav>
<Link to="/">Home</Link>|
<Link to="/addproduct">AddProduct</Link>|
<Link to="/products">Products</Link>|
<Link to="/title">Title</Link>|
<Link to="/app">App</Link>|
<Link to="/users">Users</Link>
</nav>
<div>
<Outlet />
</div>
</>);
}

export default Nav;
